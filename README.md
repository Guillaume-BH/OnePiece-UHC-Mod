# OnePiece-Mod

`OnePiece-Mod` is a simple plugin who was created solely to add a "Moderation" aspect to
organized [OnePiece UHC](https://docs.onepiece-uhc.fr/) games.

## Dependencies

This plugin depends on the library [SmartInv](https://github.com/MinusKube/SmartInvs)
Please make sure to have this plugin too.

### Building

> [Maven](https://maven.apache.org/) is required to build this plugin.

For building, use

```shell
mvn package
```

### Installing

After building, you can find the plugin in the `target` directory. The file will be named `onepiece-mod-1.0.jar`. You
can rename it to `OnePiece-Mod.jar` for easier use. To install the plugin, simply place the `OnePiece-Mod.jar` file into
the `plugins` directory of your Minecraft server. After starting or restarting the server, the plugin should be loaded
and ready to use.
